# 安装 3D 可视化依赖 - 详细步骤

## 方法一：使用 PowerShell（推荐）

1. **打开 PowerShell**
   - 按 `Win + X`，选择 "Windows PowerShell" 或 "终端"
   - 或者按 `Win + R`，输入 `powershell`，按回车

2. **导航到项目目录**
   ```powershell
   cd C:\Users\31600\Desktop\tcm-knowledge-graph\frontend
   ```

3. **安装依赖**
   ```powershell
   npm install three @react-three/fiber @react-three/drei
   ```

4. **等待安装完成**（可能需要几分钟）

## 方法二：使用命令提示符（CMD）

1. **打开命令提示符**
   - 按 `Win + R`，输入 `cmd`，按回车

2. **导航到项目目录**
   ```cmd
   cd C:\Users\31600\Desktop\tcm-knowledge-graph\frontend
   ```

3. **安装依赖**
   ```cmd
   npm install three @react-three/fiber @react-three/drei
   ```

## 方法三：在 VS Code 中执行

1. **打开 VS Code**
2. **打开终端**
   - 按 `Ctrl + ~`（波浪号键）
   - 或点击菜单：终端 → 新建终端

3. **在终端中执行**
   ```bash
   cd frontend
   npm install three @react-three/fiber @react-three/drei
   ```

## 验证安装

安装完成后，检查 `frontend/package.json` 文件，应该能看到：
- `"three": "^0.159.0"`
- `"@react-three/fiber": "^8.15.11"`
- `"@react-three/drei": "^9.88.13"`

## 如果遇到问题

### 问题1：提示 "npm 不是内部或外部命令"
- **解决**：需要先安装 Node.js
- 下载地址：https://nodejs.org/
- 安装后重启终端

### 问题2：安装速度慢
- **解决**：可以使用国内镜像
  ```bash
  npm install three @react-three/fiber @react-three/drei --registry=https://registry.npmmirror.com
  ```

### 问题3：权限错误
- **解决**：以管理员身份运行 PowerShell 或 CMD

## 安装完成后

重启前端开发服务器：
```bash
npm run dev
```

然后访问：http://localhost:3000/visualizations

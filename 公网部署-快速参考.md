# 🌐 公网部署快速参考

## 🚀 最快方式（5分钟）

### 前端 → Vercel
1. 访问 https://vercel.com，用GitHub登录
2. 点击 "New Project" → 选择仓库或上传 `frontend/dist`
3. 配置：Framework = Vite, Output = dist
4. 完成！获得地址：`https://xxx.vercel.app`

### 后端 → Railway
1. 访问 https://railway.app，用GitHub登录
2. 点击 "New Project" → 选择仓库或上传 `backend` 文件夹
3. 添加环境变量（见下方）
4. 完成！获得地址：`https://xxx.up.railway.app`

## 📝 必需的环境变量

### 前端（Vercel）
```
VITE_API_URL=https://your-backend-url.com/api
```

### 后端（Railway）
```
NODE_ENV=production
PORT=3001
NEO4J_URI=neo4j+s://your-neo4j-uri
NEO4J_USER=neo4j
NEO4J_PASSWORD=your-password
FRONTEND_URL=https://your-frontend.vercel.app
```

## ❓ 常见问题

**Q: 前端无法访问后端？**
A: 检查CORS配置和API地址

**Q: Neo4j连接失败？**
A: 检查URI格式和密码，确认防火墙设置

**Q: 静态资源404？**
A: 检查构建路径和Nginx配置

详细解决方案见 `网站公网部署指南.md`

# 更换背景图片说明

## 📝 步骤

### 方法1：直接替换文件（推荐）

1. **将新背景图片放到 `frontend/public/` 目录**
   - 文件名：`background.png`（优先）或 `background.jpg`
   - 如果使用 PNG 格式，文件名必须是 `background.png`
   - 如果使用 JPG 格式，文件名必须是 `background.jpg`

2. **图片要求**
   - 建议分辨率：1920x1080 或更高
   - 格式：PNG（推荐，支持透明）或 JPG
   - 文件大小：建议控制在 2MB 以内，以确保加载速度

3. **提交代码**
   - 将新图片提交到 Git
   - 推送到 GitHub
   - Vercel 会自动部署

### 方法2：保留旧背景作为备用

- 如果新背景图片加载失败，系统会自动尝试加载 `background.gif`
- 如果所有背景都加载失败，会使用 CSS 渐变作为备用方案

## ✅ 已更新的代码

- `Background.tsx` 现在支持多种图片格式：
  - 优先加载：`background.png`
  - 其次：`background.jpg`
  - 备用：`background.gif`
  - 最终备用：CSS 渐变

## 🎯 当前状态

代码已更新，支持新背景图片。您只需要：

1. **将新背景图片放到 `frontend/public/background.png`**
2. **提交并推送代码**
3. **等待 Vercel 部署完成**

## 💡 提示

- 如果图片很大，可以考虑压缩以提升加载速度
- PNG 格式支持透明背景，JPG 格式文件更小
- 建议使用 PNG 格式以获得最佳效果

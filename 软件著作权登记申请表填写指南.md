### 四、软件功能说明

#### 1. 软件主要功能
```
本系统是一个基于Neo4j图数据库的中医知识图谱Web应用，主要功能包括：

1. 首页仪表板模块：提供数据概览统计、系统状态展示和快速操作入口，实时显示节点总数、关系总数、数据完整度等关键指标。

2. 图谱探索模块：实现交互式知识图谱浏览，支持多层级节点展开、实时节点详情展示和多种布局算法（力导向、同心圆、层次布局等），支持节点拖拽、缩放、筛选等功能。

3. 智能搜索模块：提供全文搜索功能、高级筛选选项、搜索结果高亮和相关推荐系统，支持按关键词、代码、分类等多条件搜索。

4. 数据分析模块：实现多维度统计分析、交互式图表展示、数据导出功能和质量评估报告，包括节点分类分布、层级分布、演化趋势等分析图表。

5. 可视化模块：提供2D图谱可视化（Cytoscape.js）、3D图谱可视化（Three.js + React Three Fiber）、时间线图和演化图等多种可视化方式，支持图谱旋转、缩放、视角切换等交互操作。

6. 节点详情模块：实现节点信息展示、关联关系展示和详细信息查看功能，包括节点属性、父子关系、百科描述、关联节点等信息。

系统采用前后端分离架构，前端使用React+TypeScript开发，后端使用Node.js+Express开发，集成Neo4j图数据库，提供RESTful API接口，支持2D/3D知识图谱可视化展示。
```

#### 2. 软件技术特点
```
1. 图数据库集成：采用Neo4j图数据库作为核心存储引擎，高效存储和查询复杂的中医知识关系，支持Cypher查询语言。

2. 2D/3D可视化：利用Cytoscape.js实现知识图谱的2D平面可视化，利用Three.js + React Three Fiber实现3D立体可视化展示，提供丰富的视觉体验。

3. 智能搜索与过滤：提供强大的全文搜索功能，支持多条件筛选，快速定位所需的中医知识点，支持搜索结果高亮和相关推荐。

4. 数据分析与图表：集成ECharts等图表库，对知识图谱数据进行多维度分析和可视化，包括节点分类分布、层级分布、演化趋势等分析。

5. 前后端分离架构：前端采用React+TypeScript，后端采用Node.js+Express，保证系统的高效性和可维护性，支持独立部署和扩展。

6. RESTful API：提供标准化的API接口，便于第三方集成和扩展，支持Swagger API文档自动生成。

7. 响应式设计：适应不同设备和屏幕尺寸，提供良好的用户体验。

8. 高性能渲染：采用虚拟化技术优化大规模图谱的渲染性能，支持节点懒加载和按需渲染。
```

#### 3. 软件创新点
```
1. 知识图谱可视化应用：将传统中医知识系统化、可视化，通过图数据库和可视化技术，直观展示中医知识的复杂结构和关联关系。

2. 2D/3D多维度展示：提供2D平面图谱和3D立体图谱两种可视化方式，满足不同场景下的知识探索需求，创新性地将3D可视化技术应用于中医知识图谱展示。

3. 智能搜索与关联分析：基于图数据库的复杂查询能力，实现智能搜索和关联分析，帮助用户快速发现知识点之间的隐藏关系。

4. 模块化架构设计：采用前后端分离和模块化设计，各功能模块独立，便于维护和扩展，支持后续功能迭代。

5. 现代化技术栈：使用React、TypeScript、Neo4j等现代Web技术，代码质量高，易于维护和扩展，具有良好的技术先进性。
```

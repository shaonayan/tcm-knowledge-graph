# 🚀 快速部署说明

## Windows 一键部署

### 步骤1：运行部署脚本

双击运行 `一键部署.bat`

脚本会自动：
- ✅ 检查Node.js环境
- ✅ 安装前端依赖
- ✅ 构建前端应用
- ✅ 创建后端配置文件

### 步骤2：配置Neo4j

编辑 `backend\.env` 文件，填入您的Neo4j连接信息：

```env
NODE_ENV=production
PORT=3001
NEO4J_URI=neo4j+s://f36358f7.databases.neo4j.io
NEO4J_USER=neo4j
NEO4J_PASSWORD=您的Neo4j密码
FRONTEND_URL=http://localhost:3001
```

### 步骤3：启动服务器

双击运行 `启动生产服务器.bat`

或手动运行：
```bash
cd backend
set NODE_ENV=production
node server-simple.js
```

### 步骤4：访问应用

打开浏览器访问：**http://localhost:3001**

## 📱 部署为网站

### 本地访问
- 同一台电脑：http://localhost:3001
- 同一局域网：http://您的IP:3001

### 公网访问（需要云服务器）

1. **购买云服务器**（阿里云、腾讯云等）
2. **上传项目文件**
3. **在服务器上运行部署脚本**
4. **配置防火墙开放3001端口**
5. **访问：http://您的服务器IP:3001**

### 使用域名（推荐）

1. **购买域名**
2. **配置DNS解析到服务器IP**
3. **配置Nginx反向代理**（参考 `部署指南.md`）
4. **配置SSL证书**（HTTPS）
5. **访问：https://您的域名**

## 🔧 环境变量

创建 `backend\.env` 文件：

```env
# 运行环境
NODE_ENV=production

# 服务器端口
PORT=3001

# Neo4j配置
NEO4J_URI=neo4j+s://您的Neo4j地址
NEO4J_USER=neo4j
NEO4J_PASSWORD=您的密码

# 前端URL（CORS）
FRONTEND_URL=http://localhost:3001
```

## ✅ 验证部署

1. **健康检查**：http://localhost:3001/health
2. **API信息**：http://localhost:3001/api/info
3. **前端页面**：http://localhost:3001

## 📝 注意事项

1. **确保前端已构建**：`frontend/dist` 目录必须存在
2. **确保Neo4j可访问**：检查网络连接和密码
3. **端口未被占用**：确保3001端口可用
4. **防火墙设置**：如需公网访问，开放3001端口

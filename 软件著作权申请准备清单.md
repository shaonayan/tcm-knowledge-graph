### 开发目的
```
开发中医知识图谱可视化系统，用于中医教学、临床诊断和学术研究，提供知识图谱探索、搜索、分析和可视化功能。
```

### 面向领域/行业
```
医疗行业、教育行业、中医药研究
```

### 软件技术特点
```
基于Neo4j图数据库的中医知识图谱系统，提供2D/3D图谱可视化、智能搜索、数据分析和知识探索功能。
```

### 软件的主要功能
```
本系统是一个基于Neo4j图数据库的中医知识图谱Web应用，主要功能包括：

1. 控制台模块：提供数据概览统计、系统状态展示和快速操作入口。

2. 图谱探索模块：实现交互式知识图谱浏览，支持多层级节点展开、实时节点详情展示和多种布局算法。

3. 搜索模块：提供全文搜索功能、高级筛选选项、搜索结果高亮和相关推荐系统。

4. 分析模块：实现多维度统计分析、交互式图表展示、数据导出功能和质量评估报告。

5. 可视化模块：提供2D图谱可视化（Cytoscape）、3D图谱可视化（Three.js）、时间线图和演化图等多种可视化方式。

6. 节点详情模块：实现节点信息展示、关联关系展示和详细信息查看功能。

系统采用前后端分离架构，前端使用React+TypeScript开发，后端使用Node.js+Express开发，集成Neo4j图数据库，提供RESTful API接口，支持2D/3D知识图谱可视化展示。
```

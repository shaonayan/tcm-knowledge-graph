## 🎨 设计特点

### 透明玻璃效果
- **背景**：`rgba(0, 0, 0, 0.2-0.4)` 半透明黑色
- **模糊**：`backdrop-filter: blur(20px) saturate(180%)`
- **边框**：`rgba(255, 255, 255, 0.15-0.2)` 半透明白色
- **阴影**：多层阴影增强层次感
- **高光**：顶部1px高光线

### 文字系统
- **主标题**：`#ffffff` 白色，`text-shadow: 0 2px 12px rgba(0, 0, 0, 0.8)`
- **副标题**：`rgba(255, 255, 255, 0.7)` 半透明白色
- **正文**：`rgba(255, 255, 255, 0.8-0.9)` 半透明白色
- **辅助文字**：`rgba(255, 255, 255, 0.7)` 半透明白色

### 交互效果
- **悬停提升**：`transform: translateY(-1px to -2px)`
- **背景变深**：悬停时背景透明度增加
- **边框变亮**：悬停时边框更明显
- **平滑过渡**：所有变化都有0.3-0.4s的平滑动画

## 📁 背景文件位置

背景GIF已复制到：
```
frontend/public/background.gif
```

如果背景文件没有正确显示，请检查：
1. 文件是否存在于 `frontend/public/` 目录
2. 文件名是否为 `background.gif`
3. 浏览器是否支持GIF格式

## 🔧 如果背景不显示

### 方法1：检查文件路径
确保 `background.gif` 在 `frontend/public/` 目录下

### 方法2：使用其他格式
如果GIF太大或有问题，可以：
1. 将GIF转换为视频（MP4）
2. 使用CSS动画创建类似效果
3. 使用静态图片作为背景

### 方法3：手动设置背景
在 `globals.css` 中修改：
```css
background-image: url('/background.gif');
```
改为你的背景文件路径。

## 🎯 设计亮点

1. **完全透明**：所有UI元素都是透明的，背景清晰可见
2. **层次分明**：通过透明度和模糊程度区分层次
3. **可读性强**：白色文字 + 阴影，在任何背景下都清晰
4. **现代感**：毛玻璃效果 + 动态背景，高端大气
5. **一致性**：所有组件统一使用透明玻璃效果

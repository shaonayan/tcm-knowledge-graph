# 快速开始指南

## 🚀 启动项目

### 1. 启动后端服务器
```bash
cd backend
npm install  # 首次运行需要
node server-simple.js
```
后端将在 http://localhost:3001 运行

### 2. 启动前端应用
```bash
cd frontend
npm install  # 首次运行需要
npm run dev
```
前端将在 http://localhost:3000 运行

## 📋 主要功能页面

1. **首页仪表板** (`/`) - 查看整体统计信息
2. **图谱探索器** (`/explorer`) - 交互式图谱可视化
3. **智能搜索** (`/search`) - 搜索节点
4. **数据分析** (`/analytics`) - 数据统计图表
5. **高级可视化** (`/visualizations`) - 3D视图、时间线、演化展示

## 🔑 快捷键（图谱探索器）

- `Ctrl/Cmd + +` : 放大
- `Ctrl/Cmd + -` : 缩小
- `Ctrl/Cmd + 0` : 重置缩放
- `Ctrl/Cmd + F` : 适应视图

## 🎨 3D图谱视图布局

- **力导向**：基于节点关系的自然布局
- **球形**：均匀的球面分布
- **网格**：规则的3D网格排列

## 📝 注意事项

- 确保 Neo4j 数据库连接正常
- 后端服务器需要在后端启动
- 首次加载图谱数据可能需要几秒钟

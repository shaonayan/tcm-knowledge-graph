## 2. 开发目的
为中医病证分类标准构建数字化知识图谱平台，提供可视化探索、智能搜索、数据分析等功能，辅助教学、科研与临床决策。

---

## 3. 面向领域 / 行业
- 医疗健康（中医药）
- 教育培训（中医教学）
- 科研机构（知识发现与关联分析）

---

## 4. 功能概览
1. **首页仪表板**：实时统计节点数、关系数、数据完整度，提供快捷入口。
2. **图谱探索器**：交互式知识图谱浏览，支持多层级展开、节点详情、路径高亮。
3. **智能搜索**：关键词、代码、分类等多条件搜索，支持搜索结果定位与导出。
4. **数据分析**：类别分布、层级分布、演化趋势图表，可导出报表。
5. **可视化中心**：提供2D/3D图谱、时间线、演化图，多种布局与主题。
6. **节点详情**：查看节点属性、父子关系、百科描述、关联节点等信息。

---

## 5. 技术特点
- 前后端分离架构（React + Node.js）
- Neo4j图数据库 + Cypher查询
- Cytoscape.js 2D图谱、Three.js 3D渲染
- Tailwind CSS + Ant Design UI
- RESTful API + Swagger文档
- 支持Redis缓存、JWT认证扩展

# 安装 3D 可视化依赖

为了使用高级可视化功能（3D 图谱视图），需要安装以下依赖：

## 安装步骤

1. 打开终端/命令提示符
2. 进入前端目录：
   ```bash
   cd frontend
   ```

3. 安装依赖：
   ```bash
   npm install three @react-three/fiber @react-three/drei
   ```

4. 等待安装完成

## 依赖说明

- **three**: Three.js 3D 图形库
- **@react-three/fiber**: React 的 Three.js 渲染器
- **@react-three/drei**: 常用的 Three.js 辅助组件和工具

安装完成后，重启前端开发服务器即可使用高级可视化功能。
